📖 วิธีใช้งาน
1. วิธีบันทึก Hand ขั้นตอน ▼
ระหว่างเล่น (30 วินาที):
1. เลือก HU หรือ MULTI ด้านบน
2. กดกล่อง Hero Cards เลือกไพ่ในมือ
3. ช่อง Preflop — พิมพ์หรือกด chip แทรก เช่น H:r2.5x → V:c
4. กด street Flop / Turn / River เพื่อเปิด กด action + sizing
5. ใส่ Result +/- bb และกด [OK] [?] [X]
6. กด SAVE HAND

ช่วง Break (ใจเย็น):
กด + เพิ่มรายละเอียด ใส่ Stack, Blind, Read, Note
2. [OK] [?] [X] แตกต่างกันยังไง ▼
[OK] — ตัดสินใจถูกต้อง ไม่ต้อง review
[?] — ไม่แน่ใจ ต้องเอาไปวิเคราะห์กับ GTO Wizard หลัง break
[X] — รู้แล้วว่าผิด ต้อง review ป้องกันซ้ำ

เมื่อกด [?] หรือ [X] แอพจะถามว่า "ทำไมถึงตัดสินใจแบบนั้น?" — กรอกให้ครบ เพราะนี่คือ learning point ที่สำคัญที่สุด
3. Concept Tags ใช้ยังไง ▼
Tag บอกว่า hand นี้เกี่ยวกับ situation ประเภทไหน

ตัวอย่าง: hand ที่ hero c-bet แล้วโดน raise → tag c-bet
hand ที่ตัดสินใจ push/fold ใน short stack → tag push/fold + ICM

ประโยชน์: ดูหน้า Session แล้วจะเห็นว่า situation ไหนที่ [?] [X] เยอะที่สุด — นั่นคือ weak spot ที่ต้องไป study
4. Action Shorthand อ่านว่าอะไร ▼
H: = Hero V: = Villain → = แล้วจึง

r2.5x = raise 2.5bb 3b9x = 3-bet 9bb
c = call f = fold x = check
b50 = bet 50% pot b100 = pot bet
shv = shove all-in cb = c-bet
sq = squeeze ai = all-in

ตัวอย่าง: H:r2.5x → V:3b9x → H:4b22 → V:c
= Hero raise 2.5bb, V 3-bet 9bb, Hero 4-bet 22bb, V call
5. Villain Notes ใช้ยังไง ▼
แท็บ Villain ใช้เก็บ notes เกี่ยวกับผู้เล่นที่เจอบ่อยหรือน่าสนใจ

เพิ่ม Villain ด้วยชื่อหรือรายละเอียดที่จำได้ เช่น Seat 3 hoodie
แล้วกด + เพิ่ม note จดนิสัยที่สังเกตเห็น เช่น:
c-bet 100% ทุก flop
fold to 3-bet preflop เยอะมาก
hero call river ด้วย 2nd pair

ข้อมูลนี้ช่วยได้มากถ้าเจอคนเดิมในทัวร์ถัดไป
